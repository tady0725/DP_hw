## 安裝虛擬環境 && 創建虛擬環境
{% note [class] [no-icon] %}

- pip install virtualenv

- cd DP_hw

- python -m venv venv
- venv\Scripts\activate.bat

{% endnote %}



## 安裝相關套件
{% note [class] [no-icon] %}

- cd hw
- pip install -r requirements.txt

{% endnote %}
